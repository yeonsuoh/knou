## 5.1 월드 와이드 웹

### 1. 웹의 개요
* 문서를 하이퍼텍스트 형태로 표현하는 HTML 형식 사용
* 이를 전송할 수 있는 HTTP 통신규약을 가짐
* 인터넷: TCP/IP 라는 전송규약을 사용하여 연결된 모든 네트워크

(1) 웹의 발전
* 팀 버너스 리
* W3C

(2) 하이퍼미디어와 웹

(3) 웹 브라우저의 종류
